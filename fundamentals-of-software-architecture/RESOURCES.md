# Fundamentals of Software Architecture Resources

## Knowledge

- [Book: _Fundamentals of Software Architecture_ — Mark Richards & Neal Ford (O'Reilly, 2020)](https://www.oreilly.com/library/view/fundamentals-of-software/9781492043447/)
  Primary source. The user is actively reading this. Use as ground truth for all lessons aligned to the book.

- [Book companion site — fundamentalsofsoftwarearchitecture.com](https://fundamentalsofsoftwarearchitecture.com/)
  Author-maintained resources, diagrams, and errata.

## Wisdom (Communities)

- [r/softwarearchitecture](https://www.reddit.com/r/softwarearchitecture/)
  Practitioner discussion, case studies, architecture review threads. Good for reality-checking book concepts against real-world application.

- [Software Architecture Monday — Mark Richards (YouTube)](https://www.youtube.com/c/MarkRichards-SoftwareArchitect)
  Weekly short videos by one of the book's authors. High-trust, direct extension of the book content.
