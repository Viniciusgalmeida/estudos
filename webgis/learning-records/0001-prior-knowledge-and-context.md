# Prior knowledge established at session start

User is a React front-end developer starting a new job that involves WebGIS and satellite imagery. They have some exposure to WebGIS concepts but no deep experience. The team likely uses MapLibre GL JS (unconfirmed). They will need to display satellite maps, draw/annotate over them, and render data layers visually.

**Implications:** Start from the foundational mental model (how web maps work) rather than jumping to code. First lessons should build the vocabulary and tile-system understanding that everything else depends on. Once confirmed, lean into MapLibre-specific patterns.
