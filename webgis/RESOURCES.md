# WebGIS Resources

## Knowledge

- [MapLibre GL JS Documentation](https://maplibre.org/maplibre-gl-js/docs/)
  Official docs for MapLibre GL JS — the likely library for this job. Use for: API reference, examples, migration from Mapbox GL JS.

- [MapLibre GL JS Examples](https://maplibre.org/maplibre-gl-js/docs/examples/)
  Runnable code examples covering most common tasks. Use for: quick reference when implementing specific features (drawing, layers, sources).

- [OpenStreetMap Wiki — Tiles](https://wiki.openstreetmap.org/wiki/Tiles)
  Authoritative explanation of how web map tiles work (XYZ scheme, zoom levels). Use for: understanding the tile system that underlies all web maps.

- [GeoJSON Specification (RFC 7946)](https://datatracker.ietf.org/doc/html/rfc7946)
  The official spec for GeoJSON — the data format you'll encounter constantly. Use for: understanding Feature, FeatureCollection, geometry types.

- [Mapbox — Understanding Map Projections](https://docs.mapbox.com/help/glossary/projection/)
  Clear explanation of projections used in web mapping (Web Mercator). MapLibre forked from Mapbox so the concepts transfer directly.

## Wisdom (Communities)

- [GIS Stack Exchange](https://gis.stackexchange.com/)
  High-quality Q&A for GIS and mapping questions. Strong signal-to-noise. Use for: debugging weird behavior, understanding spatial concepts.

- [MapLibre GitHub Discussions](https://github.com/maplibre/maplibre-gl-js/discussions)
  Active community around MapLibre specifically. Use for: library-specific questions, feature requests, seeing how others solve problems.
